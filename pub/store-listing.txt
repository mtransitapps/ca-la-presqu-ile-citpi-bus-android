Full description:
BETA: Thanks for your patience while the last features are being added and the last bugs are being fixed.

This application adds CIT La Presqu'Île bus information to MonTransit BETA.

CIT La Presqu'Île serves Hudson, Vaudreuil-Dorion, Pincourt, Rigaud, Saint-Lazare, L'Île-Perrot and Notre-Dame-de-l-Île-Perrot.

Once this application is installed, the MonTransit BETA app will display bus information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit BETA app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by the Agence Métropolitaine de Transport.
http://www.amt.qc.ca/developers/

This application is free and open-source:
https://github.com/mtransitapps/ca-la-presqu-ile-citpi-bus-android

This app is not related with the Agence Métropolitaine de Transport and CIT La Presqu'Île.
